# EXPLORIA Codebase

این پوشه Codebase رسمی توسعه EXPLORIA بر پایه Laravel + React Monolith است.

## وضعیت فعلی

Governance Handoff تکمیل شده است. Bootstrap رسمی Laravel به‌دلیل محدودیت شبکه Package Manager در محیط Codex در انتظار دریافت Dependencyهاست. جزئیات در `docs/BOOTSTRAP_STATUS.md` ثبت شده است.

## قواعد الزامی

- پیش از هر تغییر، `AGENTS.md` خوانده شود.
- اسناد `docs/governance` مرجع Scope، معماری، Acceptance و Gateها هستند.
- هیچ Secret یا فایل `.env` واقعی Commit نشود.
- کدنویسی Feature پیش از تکمیل Bootstrap و اجرای تست پایه مجاز نیست.
