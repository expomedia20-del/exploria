# چک‌لیست UAT نقش‌محور پنل‌ها

این چک‌لیست برای جلوگیری از ناهمخوانی میان «نقش عملیاتی»، «نوع اکانت»، «منوی قابل مشاهده» و «پنل مقصد بعد از ورود» ثبت شده است. حساب‌های زیر حساب‌های نمایشی local/seeder هستند و برای مرور واقعی تولیدی جایگزین حساب‌های عملیاتی امن می‌شوند.

## قرارداد ورود و پنل

| نقش/حساب نمایشی | مسیر مقصد بعد از ورود | پنل قابل مرور | انتظار UAT |
| --- | --- | --- | --- |
| `admin@example.test` | `/dashboard` | داشبورد مرکزی | دسترسی کامل مدیریتی و لینک‌های نقش/دسترسی نمایش داده شود. |
| `regional@example.test` | `/dashboard` | داشبورد ادمین استانی/منطقه‌ای | داده‌ها و اقدام‌ها با محدوده منطقه‌ای کنترل شوند. |
| `viewer@example.test` | `/dashboard` | داشبورد مشاهده‌گر | مشاهده مجاز باشد، عملیات حساس مسدود بماند. |
| `demo@example.test` | `/participant/dashboard` | پنل بازدیدکننده | مسیر کاربر نهایی و مشارکت کمپین بدون دسترسی مدیریتی باشد. |
| `ravaq.manager@example.test` | `/ravaq/dashboard` | پنل مدیر رواق/هاب | فقط هاب‌ها و واحدهای تخصیص‌یافته نمایش داده شوند. |
| `venue.manager.ecopark@example.test` | `/venue/dashboard` | پنل مدیر اجرایی مکان | ورود معمولی مستقیم به پنل مکان برسد، نه داشبورد عمومی. |
| `cafe.eco@example.test` | `/partner/dashboard` | پنل فروشگاه/واحد تجاری | پاداش، پیشنهاد، پروفایل و تبلیغ همان واحد قابل کنترل باشد. |
| `ravaq.store@example.test` | `/partner/dashboard` | پنل فروشگاه رواق | محدوده فروشگاه داخل رواق از سایر واحدها جدا بماند. |
| `family.sponsor@example.test` | `/sponsor/dashboard` | پنل اسپانسر | درخواست‌ها، پاداش‌ها و تبلیغات اسپانسری قابل پیگیری باشند. |

## قرارداد منوی نقش‌ها

| نقش عملیاتی | انتظار منوی کناری |
| --- | --- |
| ادمین مرکزی، ادمین منطقه‌ای و اپراتور بدون scope اختصاصی | منوی مدیریتی کامل یا خواندنی متناسب با account role نمایش داده شود. |
| مدیر پروژه مکانی اکسپلوریا | عملیات داخلی، پنل مکان، کمپین، شرکا، QR، scan log و پشتیبانی نمایش داده شود. |
| مجری میدانی یا یاریگر گنج | نقشه عملیات کمپین، QR/scan یا پنل مشارکت‌کننده و پشتیبانی نمایش داده شود. |
| مدیر تبلیغات نمایشگرها | عملیات نمایشگرها، تبلیغات، کمپین‌ها و پشتیبانی نمایش داده شود. |
| مدیر اجرایی مکان | پنل مکان، نقشه عملیات، کمپین‌ها، شرکا، اسپانسرها، تبلیغات و پشتیبانی نمایش داده شود. |
| مدیر هاب یا رواق | پنل هاب/رواق، نقشه عملیات، کمپین‌ها، شرکا، اعضا، اسپانسرها، تبلیغات و پشتیبانی نمایش داده شود. |
| فروشگاه یا واحد تجاری | پنل فروشگاه، تبلیغات فروشگاه و پشتیبانی نمایش داده شود. |
| اسپانسر | فقط پنل اسپانسر و پشتیبانی نمایش داده شود. |
| بازدیدکننده | فقط پنل مشارکت‌کننده نمایش داده شود. |

## قرارداد محتوای داخل پنل

| پنل | CTA و فرم مجاز | مواردی که نباید داخل پنل نقش نمایش/اجرا شود |
| --- | --- | --- |
| فروشگاه / واحد تجاری | پروفایل فروشگاه، پیشنهاد پاداش یا تخفیف، موجودی، تایید مصرف پاداش و تبلیغ فروشگاه همان واحد | قرارداد اسپانسری، حمایت از مسیر، پیام اسپانسری مرکزی، مدیریت سایر فروشگاه‌ها یا تایید نهایی کمپین |
| اسپانسر | پیشنهاد حمایت، بسته پاداش/محصول، تبلیغ برند و تخصیص پیشنهادی به واحدهای اجرا | تایید نهایی کمپین، قیمت‌گذاری عملیاتی، تحویل پاداش در فروشگاه، زمان‌بندی نمایشگر یا مدیریت موجودی فروشگاه |
| مدیر هاب / رواق | مشاهده محدوده هاب، آمادگی اجرا، صف تبلیغ/پاداش و گزارش اختلاف | تغییر قرارداد اسپانسر، تغییر قواعد کمپین، مدیریت فروشگاه خارج از محدوده یا عملیات ادمین مرکزی |
| مدیر مکان | وضعیت مکان، هاب‌ها، کمپین‌های جاری، شرکا، نمایشگرها و هماهنگی اجرایی | عملیات حساس مرکزی بدون تایید، ساختاردهی نقش‌ها خارج از محدوده یا تغییر سیاست‌های کلان |

## کنترل‌های منفی الزامی

| نقش | مسیر حساس | انتظار |
| --- | --- | --- |
| فروشگاه/واحد تجاری | `/admin/access-scopes` | دسترسی ممنوع باشد. |
| بازدیدکننده | `/admin/support` | دسترسی ممنوع باشد. |
| مشاهده‌گر | `/admin/display-operations` | دسترسی ممنوع باشد. |
| اسپانسر | `/partner/dashboard` و `/partner/ads` | دسترسی ممنوع باشد و اسپانسر فقط از `/sponsor/dashboard` اقدام کند. |

## وضعیت تست خودکار

این قرارداد در تست `tests/Feature/Auth/RolePanelJourneyTest.php` پوشش داده شده است. هر تغییر بعدی در نقش‌ها، سایدبار، مقصد ورود یا Route middleware باید این تست را سبز نگه دارد.
