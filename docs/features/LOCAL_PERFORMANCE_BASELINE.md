# خط مبنای محلی کارایی EXPLORIA

تاریخ اجرا: ۱۴۰۵/۰۴/۲۸ (2026-07-19)

## دامنه و روش

- محیط: توسعه محلی ویندوز، Laravel روی `127.0.0.1:8004`
- پایگاه داده: داده دموی محلی
- روش: ۲۰ درخواست ترتیبی برای هر مسیر با `Invoke-WebRequest`
- این اعداد فقط خط مبنای توسعه هستند و آزمون بار هم‌زمان یا تأیید ظرفیت محیط production محسوب نمی‌شوند.

## نتیجه

| مسیر | درخواست موفق | میانگین | P95 | بیشینه |
| --- | ---: | ---: | ---: | ---: |
| `/up` | ۲۰ از ۲۰ | ۳۹٫۵۵ ms | ۳۵٫۲۶ ms | ۱۷۴٫۰۳ ms |
| `/` | ۲۰ از ۲۰ | ۴۰٫۰۰ ms | ۵۳٫۰۳ ms | ۵۳٫۰۹ ms |

## کنترل آمادگی دمو

فرمان `php artisan exploria:demo-readiness --json` هر ۱۷ کنترل را با وضعیت `pass` و بدون هشدار یا خطا گزارش کرد. تست سناریوی فشار دمو نیز با ۲ تست و ۲۵ assertion موفق بود.

## گام لازم پیش از انتشار

در محیط staging متصل به PostgreSQL و زیرساخت مشابه production، آزمون بار هم‌زمان برای مسیر ورود OTP، اسکن QR، تکمیل مأموریت و داشبورد اجرا و معیارهای ظرفیت، نرخ خطا و P95 ثبت شود.
