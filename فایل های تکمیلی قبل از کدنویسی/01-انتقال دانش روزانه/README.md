# EXPLORIA - انتقال دانش روزانه

این پوشه مرکز تحویل و ادامه پروژه اکسپلوریا بین اکانت‌های Codex، سیستم‌ها یا نشست‌های مختلف است.

## شروع هر نشست

ابتدا این فایل‌ها را بخوانید:

1. `START_OF_DAY.md`
2. `PROJECT_STATE_2026-06-24.md`
3. `SESSION_LOG_2026-06-24.md`
4. `END_OF_DAY.md`

## مسیر معتبر فعلی

مسیر کاری معتبر پروژه:

`E:\exploria-codebase-current`

پوشه قدیمی `exploria-codebase` داخل مسیر منابع فارسی ناقص است و نباید مبنای توسعه قرار بگیرد.

## قواعد ثابت

- قبل از تغییر کد یا اسناد، بکاپ تاریخ‌دار بسازید.
- در پایان هر روز، وضعیت، تست‌ها، تغییرات و blockerها را ثبت کنید.
- UI پروژه فارسی و RTL بماند.
- دامنه پیاده‌سازی در Laravel + React monolith حفظ شود.
- `vendor` و `node_modules` معمولا وارد بکاپ سورس نشوند، مگر برای snapshot کامل وابستگی‌ها.
