# چرخه پرزنت دمو EXPLORIA - 2026-06-24

این فایل برای ارائه دمو به هیئت‌مدیره آماده شده است.

## شروع

از این آدرس شروع کنید:

```text
http://127.0.0.1:8000/board
```

جمله شروع:

```text
برای اینکه دمو شبیه ورود به سایت اصلی دیده شود، ارائه را از ورودی رسمی اکسپلوریا شروع می‌کنیم.
```

## نکته کلیدی

```text
MVP فعلی: QR، ورود موبایلی، رضایت‌نامه، ثبت بازدید، داشبورد و QR Registry.
Demo Vision: پنل‌های کامل، گنج‌ها، پاداش‌ها، فروشگاه‌ها، رواق، تبلیغات مستقل و اسپانسرها با داده ماک.
```

## جمله برای صفحه اکوسیستم

```text
این صفحه تصویر کامل محصول است. این بخش فعلاً mock frontend است تا نقش‌ها و مدل کسب‌وکار دیده شود، اما مسیر QR و داشبورد بخش MVP واقعی‌تر هستند.
```

## جمله برای تبلیغات مستقل

```text
علاوه بر کمپین‌ها، فروشگاه‌های عضو، برندهای غیرعضو و اسپانسرها می‌توانند درخواست تبلیغ ثبت کنند. انتشار تبلیغ با تایید مدیر رواق یا ادمین انجام می‌شود و برای نمایشگر ثابت، نمایشگر سیار، نقشه، صفحه پاداش یا لحظه بعد از ماموریت قابل برنامه‌ریزی است.
```

## پایان‌بندی

```text
اکسپلوریا فقط یک صفحه QR یا یک بازی ساده نیست؛ یک لایه تجربه، داده و درآمد روی فضای اکوپارک است. MVP فعلی مسیر اصلی را نشان می‌دهد و mock ecosystem تصویر محصول کامل را برای توسعه فاز بعد روشن می‌کند.
```
