# بسته انتقال اسناد EXPLORIA به پروژه کدنویسی

این پوشه مرجع انتقال کنترل‌شده اسناد مصوب از فضای «آمادگی پیش از کدنویسی» به Codebase رسمی EXPLORIA است.

## قاعده منبع واحد

- اسناد اصلی در محل فعلی خود، منبع حقیقت هستند.
- فایل‌های داخل این بسته نسخه انتقالی و فقط‌خواندنی آن اسناد محسوب می‌شوند.
- اسناد داخل این بسته نباید مستقیماً ویرایش شوند.
- پس از تغییر هر سند مصوب، `Sync-Handoff.ps1` باید اجرا شود.
- فایل `MANIFEST-SHA256.csv` نسخه، مسیر و هش محتوای منتقل‌شده را ثبت می‌کند.

## استفاده در پروژه کدنویسی

در زمان ایجاد Codebase رسمی، محتوای این پوشه به مسیر `docs/governance` منتقل می‌شود و فایل `AGENTS.md` باید در ریشه Codebase نیز قرار گیرد.

## محدودیت همگام‌سازی

همگام‌سازی یک‌طرفه است: Source Documents → Handoff Package. تغییر مستقیم نسخه‌های انتقالی مجاز نیست و در Sync بعدی بازنویسی می‌شود.
