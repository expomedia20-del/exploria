# EXPLORIA Portable Toolchain

ابزارهای قابل‌حمل توسعه در پوشه نادیده‌گرفته‌شده `.toolchain` نگهداری می‌شوند و بخشی از Source Code یا Backup اسناد نیستند.

برای فعال‌سازی در یک ترمینال PowerShell:

```powershell
$script = [IO.File]::ReadAllText('.\tools\Use-ExploriaToolchain.ps1', [Text.Encoding]::UTF8)
& ([scriptblock]::Create($script))
```

نسخه‌ها باید در سند Toolchain Readiness ثبت شوند. PHP، Node و Composer فقط از منابع رسمی دریافت می‌شوند.
